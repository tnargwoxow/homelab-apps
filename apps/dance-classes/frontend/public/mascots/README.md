# Custom mascot images

Drop your own images here and reference them from `frontend/src/lib/themes.ts`
as `'/mascots/<filename>'`. They'll be served by the backend's static-file
handler at `/mascots/<filename>`.

Suggested names (anything works — these match the comments in `themes.ts`):

- `channing-1.jpg`, `channing-2.jpg`     — heels theme mascots
- `flying-steps-1.jpg`, `flying-steps-2.jpg` — hiphop theme mascots

Tip: 320x320px PNG/JPG with a transparent or themed background looks best.
